# VUE Boilerplate
> created by: babangadi

Setup include:
- es6
- lint
- pug
- sass
- css extract
- css minify
- copy image

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
